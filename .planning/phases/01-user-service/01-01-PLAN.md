## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01 | I | auth endpoints | mitigate | Validate inputs with Pydantic; use parameterized DB queries via SQLAlchemy to prevent SQL injection. |
| T-02 | S | JWT secrets | mitigate | Store JWT secret in env var; rotate; use sufficiently long secret and HS256 or use asymmetric keys. |
| T-03 | T | Password handling | mitigate | Hash passwords with bcrypt; never log raw passwords; enforce minimal password strength. |
</threat_model>
